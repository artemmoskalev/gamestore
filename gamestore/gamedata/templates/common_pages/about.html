RESTful API (0-100 points)

Our site has 2 restfull apis. One for public use (can be used to get public game data like games by category, name, publisher and etc) and one for developer usege that can be used to get information of developers sales. Developer rest api is authenticated by api key found on developers overview http://wsdgamestore.herokuapp.com/developer/

By: Esko

Demonstration:
Public api can be accessed on http://wsdgamestore.herokuapp.com/api/games.json
Its possible parameters are: name, exactname, category, publisher, orderby = (name, category, price), order = (reverse), showmaxamount
Some examples:
http://wsdgamestore.herokuapp.com/api/games.json?callback=mycustomcallback&category=adventure
http://wsdgamestore.herokuapp.com/api/games.json?callback=mycustomcallback&category=adventure&publisher=dev2&showmaxamount=5
http://wsdgamestore.herokuapp.com/api/games.json?callback=mycustomcallback&orderby=name

Developer api can be accessed on http://wsdgamestore.herokuapp.com/api/developer.json
For developer api get request you need to allways pass username (=your username) and key (= api key from developer panel overview). Other parameters are year and order.
Some examples:
Get years 2014 sales for user dev1 (games by revenue)
http://wsdgamestore.herokuapp.com/api/developer.json?callback=mycustomcallbac&user=dev1&key=UR4QsKbvGnxdfpJ4ArX6WRHSHk4GHHJ3KGaDju33ku23unT8kXKR9AUwmVHY9bz5&year=2014&order=reverse
Get all sales for user dev1 (games by revenue)
http://wsdgamestore.herokuapp.com/api/developer.json?callback=mycustomcallbac&user=dev1&key=UR4QsKbvGnxdfpJ4ArX6WRHSHk4GHHJ3KGaDju33ku23unT8kXKR9AUwmVHY9bz5
